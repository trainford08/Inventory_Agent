Complete-profile CTA options

Looking for ways to make the primary "Start review" / "Continue" button distinct from everything else on the page (the section-card CTAs, the Delegate icon, the back-link).

Today: the hero CTA and each section card's CTA both use the same dark bg-ink fill. They're indistinguishable, so the champion has no spatial cue about which is the "big" action.
Option A
Primary purple — same as the brand

Matches the brand primary. The section-card CTAs stay dark, so the hero clearly reads as "the big one." Lowest risk; minimal change; still aligned with everything else in the app (sidebar active state, review chevron pills, etc.).
